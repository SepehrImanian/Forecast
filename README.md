# Examples

for version of app:

```bash
outside version
```

run outside for show help menu:

```bash
outside
```
or run **--help** or **help** to do that:

```bash
outside --help
```

```bash
outside today --help
```

for get data from forecast:

```bash
outside forecast
```

and about today data:

```bash
outside today
```

for speciation location using **--location** or **-l**:

```bash
outside today -l somewhere
```
